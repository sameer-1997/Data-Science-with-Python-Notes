How to start Jupyter Notebook?
===================================

1. Create a Folder where you will maintain all the programs

ew.g. E:\DataScienceSL\12Aug2019DataScience

2. Open Anaconda Prompt

Start > Type ' Anaconda Prompt' > Click the same

3. Navigate to the root folder where all your programs will be maintained

E:
cd E:\DataScienceSL\12Aug2019DataScience

4. Will start Jupyter Server instance

jupyter notebook
